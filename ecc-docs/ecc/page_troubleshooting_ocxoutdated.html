ECC Documentation - Troubleshooting - OCX outdated

    • ERROR MESSAGE:

      failed to load control '????? 'from ???.OCX. your version of ???.OCX may be outdated. make sure your using the version of the control that was provided with your application."
       

1. About

  • This error message will popup when old OCX components are installed on your system, please update these!, we have provided the most recent OCX components with our tools, but VB6 applications (like the tools) search automatically in 'windows\system32' first!

2. How to solve it?

  • Copy all OCX files from 'ecc-tools' to 'windows\system32', to backup the old OCX files, you could give them the .BAK extension.
