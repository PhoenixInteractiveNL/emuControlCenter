ECC Documentation - Troubleshooting - File is hacked

    • ERROR MESSAGE:

      'This file is hacked :('
       

1. About

  • This message will popup when internal validation found a 'difference' in the file, the file may be sabotaged or modified, so the program HALTS itself to prevent damage to your system.

    NOTE:
  • This error can occur when running on a system with a foreign charset!, we fixed the the chinese / japanese conflict in this validation, this hopefully fixes also all other charsets.

2. How to solve it?

  • Download the original files from:
