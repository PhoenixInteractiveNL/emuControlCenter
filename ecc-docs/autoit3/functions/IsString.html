Function Reference

IsString

Checks if a variable is a string type.

IsString ( variable )

 

Parameters

variable The variable/expression to check.

 

Return Value

Success: Returns 1.
Failure: Returns 0 if expression is not string type.

 

Remarks

None.

 

Related

IsArray, IsFloat, IsInt, IsNumber, IsBool, IsHWnd, IsBinary

 

Example


$x = IsString("foo")
