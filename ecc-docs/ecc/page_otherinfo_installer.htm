1. About

  • This installer was created and compiled with NSIS, a free installer creation tool by Nullsoft.

2. The BUILD string

  • There is a 'build' string in the installer, here you can 'see' the following information:
     
  • XXX.000000.00000.0000.000000.000 ECC Version
    000.XXXXXX.00000.0000.000000.000 ECC Release date
    000.000000.XXXXX.0000.000000.000 Last update
    000.000000.00000.XXXX.000000.000 Installer version
    000.000000.00000.0000.XXXXXX.000 Installer build date
    000.000000.00000.0000.000000.XXX NSIS version
