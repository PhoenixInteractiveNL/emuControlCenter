This package is an emuControlCenter eccUser folder for 'Sega Mega-CD (smcd)'

If you want use this package,
download emuControlCenter (ECC) here http://ecc.phoenixinteractive.nl/
and add this folder to the eccUser Folder within the emuControlCenter installation.
