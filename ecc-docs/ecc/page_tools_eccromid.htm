1. About

  • With ROMid, you can identify your ROMS, just like the GOODxxx tools can, but this is a 'All-In-One' tool.

    NOTE: The NES ROMS with the DiskDude! header are ignored!

2. How to use ROMid?

  • 1. Start ECC Romid (ecc-tools\eccRomid.exe).
    2. Select a rom by pressing the button 'SELECT ROM'.
    3. Select a rom and press 'OK'.
    4. ECC ROMid will now search its databases...
    5. Result will be displayed!.



    6. You can rename the rom to a 'GOOD' name by pressing the button 'RENAME'.
    7. Exit ECC ROMid.
