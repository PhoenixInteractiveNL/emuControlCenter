1. About

  • ECC Live!, is an ECC update tool, it directly applys fixes/patches/addons for ECC.

2. How to update ECC?

  • 1. Close ECC
    2. Start ECC Live! (ecc-tools\eccLive.exe).
    3. Wait until the update list has been loaded.



    4. Press the button 'update' to update ECC (if there are updates available)
    5. Start ECC
