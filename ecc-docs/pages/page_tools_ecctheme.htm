1. About

  • With ECC Theme Select, you can change the looks of ECC by giving it another theme, these theme's are files that change the icons & graphical things.

2. How to select a theme with Theme Select

  • 1. Close ECC
    2. Start ECC Theme select (ecc-tools\eccTheme.exe).
    3. Wait until the theme list has been loaded.
    4. Select an theme from the list/



    5. Press the button 'select' to apply the selected theme (if all necessary files are present).
    6. Press the button 'ECC' to close Theme Select and start ECC.
     

3. Getting more themes for ECC

  • Theme packs can be downloaded from the ECC website
    [here]

    How to install?
    1. Download the RAR package.
    2. Decompress (using winrar) in your ECC 'root' folder.
        (paths are already included in the packages)
