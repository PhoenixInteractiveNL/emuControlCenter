Function Reference

TimerDiff

Returns the difference in time from a previous call to TimerInit().

TimerDiff ( timestamp )

 

Parameters

timestamp timestamp returned from a previous call to TimerInit().

 

Return Value

Returns the time difference (in milliseconds) from a previous call to TimerInit().

 

Remarks

None.

 

Related

TimerInit

 

Example


$begin = TimerInit()
sleep(3000)
$dif = TimerDiff($begin)
MsgBox(0,"Time Difference",$dif)
