Polaroid Gallery FX version 1.0.7.8486

Hello and thank you for using Polaroid Gallery FX. By downloading our FlashXML product you have become part of the FX community and we are very pleased to welcome you and wish you a long-lasting FlashXML experience.

The Polaroid Gallery FX's features are:
	- the text is HTML/CSS formatted
	- grid, random or desk layout of the images
	- rotation and expanding scale properties for images
	- properties for effects like shadow and blur
	- many background and shade features
	- different settings for image padding, background and corners radius
	- many other particular properties on the Live Demo

You can install our component on several platforms, from simple HTML sites to Flash projects. We provide installation instructions for multiple platforms, including video tutorials.
- for HTML: www.flashxml.net/polaroid-gallery.html#swmi-free-install
- for Flash: www.flashxml.net/polaroid-gallery.html#swmi-free-flash
- for WordPress: www.flashxml.net/polaroid-gallery.html#swmi-wordpress
- for Joomla: www.flashxml.net/polaroid-gallery.html#swmi-joomla
- for Facebook: www.flashxml.net/polaroid-gallery.html#swmi-facebook

By downloading our Polaroid Gallery FX you have access to one of the most customizable Flash component on the internet. All of our components are XML driven, so there is no need to use Adobe Flash to change the component's attributes or its assets.
The easiest way to start customizing our product is with its LiveDemo, which has all the component's attributes available.
To read more about how Flash components work with XML files, and get a good understanding of how our components work please visit this FAQ: www.flashxml.net/short-explanation-on-how-fx-components-read-xml-files.html
Most of our users want to use our component from a different folder than the location of the HTML page. To see how to embed a component that's in a different folder than your page please read www.flashxml.net/path-issues-with-swf-files-configured-via-xml.html
Since most of our users need to use special characters in their text fields we have created a special FAQ just for this: www.flashxml.net/special-characters-for-fx-components.html

Answers to the most common questions and problems of our users are also provided on the FAQ section from our website. You can find it here: www.flashxml.net/faq/.

This is a list with all the product's attributes and a short description about what each attribute does:
componentWidth - gallery's width
componentHeight - gallery's height
preloader - loading animation
closeButton - close button image
limitMovement - limit the images movement within the gallery's size
positioning - images' positioning type
spacing - distance between images (only when 'grid' or 'desk' positioning are used)
maximize - enlarge images on click
expandScale - images' expanding size
bgColor1 - image frames' background color
bgAlpha1 - image frames' opacity
bgCornerRadius1 - image frames' corner adjustment
bg1PaddingH - image frame's horizontal padding
bg1PaddingTop - image frame's top padding
bg1PaddingBottom - image frame's bottom padding
bgColor2 - images' background color
bgAlpha2 - images' background opacity
bgCornerRadius2 - images' background corner adjustment
bg2PaddingH - image background's horizontal padding
bg2PaddingV - image background's vertical padding
textDistance - distance between text and image
rotationOffset - image rotation's adjustment
friction - item's friction adjustment
width - text field's width
height - image's height
cornerRadius - text fields' corner adjustment
styleSheet - text formatting set
bodyDistance - distance between title and description text
headBgColor - background color of title text field
bodyBgColor - background color of description text field
bgAlpha - background opacity of text fields
borderColor - border color for text field
borderAlpha - border's opacity
borderWidth - text fields' border size
paddingLeft - left padding of text
paddingRight - right padding of text
paddingTop - top padding of text
paddingBottom - bottom padding of text
useDropShadow - use shadow on images
distance - shadow's distance from the images
angle - shadow's direction adjustment
color - shadow's color
alpha - shadow's opacity
blurAmount - shadow's blur intensity
quality - shadow's quality
strength - shadow's size
fillType - background type
backgroundImage - background image
backgroundFitting - background fitting
backgroundColor1 - background color when image background type not used
backgroundColor2 - background color when linear or radial background type used
backgroundAlpha - background opacity
bgCornerRadius - background corners' adjustments
gradientRotation - background colors rotation
gradientRatio - background colors adjusment
gradientPosX - background colors' horizontal position
gradientPosY - background colors' vertical position
useShade - use over mask
shadeColor - mask color
shadeAlpha - mask opacity
shadeRatio - mask adjustment
shadeWidth - mask width
shadeHeight - mask height
