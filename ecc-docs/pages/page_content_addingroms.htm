Adding ROMS to ECC

  • In this example you can see how ROMS are being added to ECC, this example shows how to add Atari 2600 ROMS into ECC

    1. Click with the right mouse button on the 'platform list' on the left side of your screen, and select 'Add new ROMS'.




    2. A file browser appears, select the folder where the Atari 2600 ROMS are located, and press 'ok'.


  • 3. ECC wil now parse the ROMS into the database, you can view it's progress on top of the screen.
     


  •
    4. Close the 'status detail area', now your ROMS are added to the ECC database, and can be selected from the list.
