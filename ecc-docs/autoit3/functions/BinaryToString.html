Function Reference

BinaryToString

Converts a binary variant into its ANSI string representation.

BinaryToString ( expression )

 

Parameters

expression An expression to convert into a string.

 

Return Value

Returns a string.

 

Remarks

Unlike String() which returns a hexidecimal representation of binary data, this function will assume the binary data is an ANSI string and convert it appropriately.

 

Related

Binary, IsBinary, String

 

Example


$buffer = Binary("ABCD")

MsgBox(4096, "String() representation" , $buffer)
MsgBox(4096, "String() representation" , String($buffer) )

$buffer = BinaryToString($buffer)
MsgBox(4096, "BinaryToString() representation" , $buffer)
